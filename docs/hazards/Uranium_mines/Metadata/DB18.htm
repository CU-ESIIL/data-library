Description:

Abstract:

This is a database of abandoned Uranium mines referred to as the New Mexico Mines Database.

Purpose:

The purpose of this data set is to become a part of a growing compendium of abandoned Uranium mine data for the Midwestern and Western United States.

Completeness_Report:

The file is only complete with respect to the source files provided.

Lineage: This database is based partially on mine locations identified under the DOE NURE program.  It has been supplemented over the years by the work of Dr. Virginia McLemore and staff of the New Mexico Bureau of Geology and Mineral Resources, a division of New Mexico Institute of Mining and Technology.

Process_Step: Mostly paper files were converted to electronic files in Access format.  These were quality assured by local area experts against USGS 24K quadrangles.

Process_Description:  Access files plud data dictionary were received by ICF Consulting.  The table called uranium_mines was converted to a MapInfo file, using coordinates provided to create spatial point file.  Tabular data were converted to DBF and Access formats, after ICF_ID assignments were made.
