Title: DB17: Texas Department of Health

Process_Step: Data were received as a hardcopy fax list of highway directions to mine locations.  Locations were mapped using Delorme StreetMap product.  Maps of a subset of the 30 locations were faxed back to data provider for QA confirmation.  Distances between actual locations, according to data provider, and to locations initially mapped using Delorme were compared and found to be all within 100 feet of each other, except for one mine which was incorrectly mapped with Delorme by a half-mile.

Process_Description: Mine locations were located using Delorme software, assigned coordinates, and imported into the GIS environment using MapInfo.  DBF and Access versions of the tables were created and unique ICF_Ids assigned.
